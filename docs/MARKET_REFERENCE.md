# Referencia de mercado

Para definir OfficeSpace Advisor se analizaron plataformas de coworking y oficinas flexibles como referencia conceptual, incluyendo WeWork, Hotdesk y Regus.

## Patrones observados

- Interfaces limpias con navegacion simple.
- Uso de tarjetas para presentar espacios y metricas.
- Paletas corporativas con fondos claros, azul oscuro, grises y acentos de color.
- Busqueda orientada a disponibilidad, ubicacion, capacidad y recursos.
- Experiencias que reducen friccion para reservar espacios de trabajo flexibles.

## Tipos de espacios comunes

- Salas de juntas.
- Hot desks.
- Salas de entrevistas.
- Salas de capacitacion.
- Salas de conferencias.
- Espacios colaborativos.

## Aplicacion en el proyecto

La referencia se uso para definir datos semilla, atributos de espacios, filtros conversacionales de Alpha Assistant y el enfoque del dashboard de negocio. La solucion busca sentirse profesional y cercana: ayuda al colaborador a encontrar un espacio adecuado y entrega datos utiles para tomar mejores decisiones de ocupacion.
