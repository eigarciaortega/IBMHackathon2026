# OfficeSpace

Plataforma web para la gestión híbrida de espacios de trabajo (salas de juntas y
*hot desks*): búsqueda de disponibilidad, motor de reservas sin solapamiento y
panel de administración. Construida con microservicios en Go, PostgreSQL y una SPA
en React.
